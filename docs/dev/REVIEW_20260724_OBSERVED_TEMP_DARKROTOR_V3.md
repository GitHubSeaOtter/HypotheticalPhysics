## 1. 原仮定者裁定の実装まとめ

| 裁定 | 実装 |
|---|---|
| トイモデル段階ではスピン役割分離を廃止。粒子編集からも削除 | **sFrame/sTherm/sRep を全撤去**(エンジン・validatePreset・粒子編集・SYSTEM_PROMPT・QA)。エンジンの力学は **v1.29 と同一へ復帰**(単一 spin が A8/A9/A10 を担う)。QA `obs.equivalence` が bit 等価を検証 |
| スピンとは別に「観測温度」を用意。絶対温度に比例・係数で調整 | **obsT(観測温度係数・0〜1000・既定1)**: 観測温度 **T_obs = obsT·(1−lightSweep)·T**(T=Is² は A9 の絶対温度)。表示系(温度色・ヒストグラム・スペクトル・HUD)を全て T_obs へ切替 — **エンジンは絶対温度のまま**(観測層の分離) |
| 原子スケールでは観測温度=0 で近似 | 粗視化解釈の補強として PHYSICS.md に登録(温度は統計量 — トイの粒子は粗視化天体なので既定1) |
| コアの温度比率を廃止。コアの観測温度比率を追加 | **coreTR 廃止 → coreOT(コア観測温度比率)**: Tc_obs = coreOT·(1−lightSweep)·(Ic·sc²)。第21便の 4-52(基準問題)は観測温度への再定義で**クローズ** |
| 粒子の半径とコアの半径比率を追加 | **radius**(0.5〜500・既定 radiusScale·√\|m\|)= R の明示指定(接触・慣性・引きずり減衰・温度・扁平率に一貫使用)/ **coreRR**(0.02〜1)= Rc = coreRR·R(旧4-43 の coreRR 案が実現) |
| 表示の粒子サイズは現状と半径準拠を切替 | 設定に「粒子サイズ表示」(現状=30px頭打ち/半径準拠)を新設(hp_sizemode 永続・A/B両画面対応) |
| 半径は扁平率の計算にも使用。扁平率は粒子選択時に表示 | **扁平率 f = 5s²R³/(4Gm)**(マクローリン弱回転近似)を選択粒子の HUD に表示 |
| ダークローターに「光を掻き出す」パラメータ | **lightSweep(0〜1・既定0)**: 高速スピンコアが自星の光を掻き出して外に出さない(第19便実験Aの天体側表現)。①T_obs=0(観測温度を計測できず見掛けは冷たい)②**E11 放射冷却も (1−lightSweep) 倍**(光が出ない=冷えない)。**自星だけ隠れれば良い** — 他星の光線・レンズには介入しない(E8R 無変更で仕様を満たす — 第21次 P1「光学未対応」への回答にもなる) |
| ダークローター=高速スピンのコア+圧倒的重量の外殻 | v3 構造(§3)— v2 の質量配分(コア9割)を**反転**(コア1割=高速・外殻9割=重量) |
| コアの掻き出しは自星のみ・外殻の引きずりは銀河スケール | コアは小半径(coreRR=0.3)で局所・外殻は radius=45 の明示指定で引きずり到達を銀河スケール化 |
