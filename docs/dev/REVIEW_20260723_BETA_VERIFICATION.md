## 3. 同便の実装(4-31 多層土星スプリント第1弾)

1. **💍「土星の環(多層)」**(id: saturnMulti・beta 先行)— 実環の C(薄 n50・108–134)/
   B(濃 n170・136–182)/ **カッシーニ間隙(182–198・空)**/ A(n80・198–218)を
   R_中心=100 相当へ概算スケール。physics は 🪐(実験)の較正値を継承(kFrame=1・muF=0.2・
   γN=0.15・kappaS=0.08・Kt=3600)。共鳴衛星なしの統制実験(現実の間隙維持は Mimas 2:1 共鳴)
2. **実測較正の発見(決定論・2回再現確認済み)**: 間隙幅(A環p5−B環p95)は
   **t0=19.2 → t150=15.9 → t300=4.9 → t600=−18.2(閉鎖)** — A環が E6′ で強く内側漂流
   (中央 207→175)する一方 B環外縁は粘性拡散で +5。**「維持機構(共鳴)なしでは間隙は
   閉じる」**を見せる教材として説明文を再設計(閉じる過程が主張)。帯順序 C<B<A・
   帯内 99.7%・落下 0・散逸 0 は t600 まで保持。初期ケプラー配置による**集団呼吸
   (コヒーレント・エピサイクル周期 ≈300)**が中央値に±14 の振動として乗る(記録)
3. **QA `behavior.saturnMulti`**(!FAST)— 帯メンバーシップ(bodies 順で粒子番号が帯に固定)
   の percentile 計測で E6′ 漂流・集団呼吸に不変な判定: t150 間隙≥8(残存)・t600 間隙<8
   (閉鎖)・帯順序保持・安定。※当初の移動窓方式は環の外側の空白を誤検出するため不採用
4. **hp_beta_* キー分離の再裁定**(4-31 起動条件): 本スプリントは**プリセット追加のみで
   セーブスキーマ非破壊** → 分離は**今回も見送り**(破壊的スキーマ実験を行う時点で再々検討)
