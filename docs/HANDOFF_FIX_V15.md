## F1: E1 の Plummer 型統一(L5 の解消)— P0

**確定式**: `w_j(r) = m_j / √(d_j² + ε²)`(E1′)。∇w_j = −m_j·(r−r_j)/(d²+ε²)^{3/2} となり
**E4 = G·∇W が厳密に成立**する。ε→0 で原仮定 w=m/d に一致(A3 の文言と整合)。
遠方漸近・d=0 の値(m/ε)は旧式と同一。中間域で W が最大 +41%(d=ε 時)増えるため、回帰は挙動指標でゲートする。
なお traceRay の `nval`(屈折率 n)は既に Plummer 型であり(L963)、本修正で n・∇n・W・E4 の4者が単一の場に統一される。

各差分は str_replace 可能な一意アンカーで示す(v1.4 402df57 の行番号は参考)。

## F2: エネルギー帳簿の完全化(L3 の真の閉性)— P0

方針(裁定 #4): A9「スピン=熱」は温存し、**散逸の行き先を A12 の放射帳簿 radE に厳密計上**する。
A12 を A12′ に拡張:「接線摩擦のすべり散逸と、スピン同期(E10′)で失われる回転エネルギーも光として放射され radE に計上される」。
P・L の経路には一切触れないため V1 は影響を受けない。

**(a) E9 接線摩擦のすべり散逸(L744-751 のブロックを置換)**
```js
// 旧
            const jt=muF*vs/(3*K);            // Δv_s = −muF·v_s(muF≤1 で無条件安定)
            vx[i]-=jt*invMi*txv; vy[i]-=jt*invMi*tyv;
            vx[j]+=jt*invMj*txv; vy[j]+=jt*invMj*tyv;
            if(!pinned[i]) spin[i]+=Li*jt/(0.5*mi*Ri*Ri);  // ds = +R̃·j_t/I, I=½mR²
            if(!pinned[j]) spin[j]+=Lj*jt/(0.5*mj*Rj*Rj);
// 新
            const jt=muF*vs/(3*K);            // Δv_s = −muF·v_s(muF≤1 で無条件安定)
            // v1.5(A12′): すべり散逸の帳簿。変更されるDOF(接線速度成分+スピン)のみの運動エネルギー差を厳密計上
            const vti0=vx[i]*txv+vy[i]*tyv, vtj0=vx[j]*txv+vy[j]*tyv;
            const Ii9=0.5*mi*Ri*Ri, Ij9=0.5*mj*Rj*Rj;
            const si0=spin[i], sj0=spin[j];
            vx[i]-=jt*invMi*txv; vy[i]-=jt*invMi*tyv;
            vx[j]+=jt*invMj*txv; vy[j]+=jt*invMj*tyv;
            if(!pinned[i]) spin[i]+=Li*jt/Ii9;             // ds = +R̃·j_t/I, I=½mR²
```
